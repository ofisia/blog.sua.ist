Setelah beberapa kali belakangan ini mendapati bahwa daftar tautan dalam aplikasi [Folio](https://savewithfolio.com/) tidak saling [*sync*](https://en.wikipedia.org/wiki/Data_synchronization) antar perangkan ketika saya beri *tag* atau saya hapus, akhirnya hari ini saya putuskan untuk kembali menggunakan [Instapaper](https://instapaper.com/). Sebenarnya lebih banyak fitur menarik pada Folio ketimbang Instapaper, akan tetapi berkali-kali membaca ulang sejumlah artikel yang sejatinya telah saya hapus lewat perangkat lain membuat saya jadi overdosis *déjà vu*.

(sumber gambar [kiri](https://savewithfolio.com) dan [kanan](https://instapaper.com))

Saya sudah mencoba memeriksa silang [Play Store](https://play.google.com/store/apps/details?id=co.lessisbetter.folio), [App Store](https://apps.apple.com/us/app/folio-save-now-read-later/id6746292066), [blog](https://savewithfolio.com/blog/) serta [*help center*](https://support.savewithfolio.com/en/) resmi Folio dan dapat saya pastikan bahwa belum ada lagi pemutakhiran oleh bung [Nick Chapman](https://www.threads.com/@nchapman) sejak tahun dua ribu dua puluh lima kemarin. Namun akan saya tetap pantau sesekali barangkali bakal ada perubahan lebih baik, sebab menurut saya Folio memang lebih memiliki potensi untuk menggantikan—atau lebih tepatnya mereplikasi—[Pocket](https://getpocket.com/home) dibanding Instapaper.
